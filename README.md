# 🍳 Игра «Напиши слово»

Обучающая игра для детей 3-5 лет. Помогает учить буквы и читать!

## 🎮 Как играть

1. Выбери уровень сложности
2. Посмотри на картинку
3. Напиши слово по буквам
4. Проверь себя!

## 🚀 Играть онлайн

[👉 Открыть игру](https://твой-ник.github.io/имя-репозитория/)

## 🛠 Технологии

- HTML5
- CSS3 (анимации)
- Vanilla JavaScript
- Web Audio API
